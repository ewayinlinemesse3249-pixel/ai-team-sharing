# Real Prompt Sample

This file is intentionally sanitized for the public repository.

The original local prompt sample contains internal project names, people names, Jira keys, and private links. It was not uploaded to avoid exposing private information.

The public deck only uses the anonymized lesson extracted from that prompt:

- Define the input contract.
- Preserve the Jira CSV header exactly.
- Map fields with explicit rules.
- Filter invalid rows and deduplicate records.
- Validate output before delivery.
- Turn the repeatable process into a skill.
